# Cursor Usage by tomippe

仓库: [tomippe/cursor-usage](https://github.com/tomippe/cursor-usage)

[wrick17/cursor-metrics](https://github.com/wrick17/cursor-metrics) 的分支（MIT）。

在状态栏显示 Cursor 套餐使用量，面向按支出计费的套餐（Ultra / Pro）：

- 套餐名称（例如 `Ultra`）
- 定额用量 / 原厂模型 / API 使用百分比
- 按需（额外）支出

若 API 返回旧版请求配额数据，仍会显示包含的请求数。

**扩展 ID:** `tomippe.cursor-usage`

## 上游

原项目: [Cursor Usage / cursor-metrics](https://github.com/wrick17/cursor-metrics)（Pratyush Poddar / wrick17）。本分支保留 MIT 许可证与原始版权声明。

与 Anysphere / Cursor 无关联。

## 命令

- `Cursor Usage: 打开仪表盘`
- `Cursor Usage: 显示详情`
- `Cursor Usage: 刷新`

## 设置

与上游相同的 `cursorUsage.*` 配置项（若同时安装原扩展会冲突，请卸载其中一个）。
