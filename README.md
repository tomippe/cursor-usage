## English

**Official page:** [apps.tomippe.jp/cursor-usage](https://apps.tomippe.jp/cursor-usage/)  
Repository: [tomippe/cursor-usage](https://github.com/tomippe/cursor-usage)

Fork of [wrick17/cursor-metrics](https://github.com/wrick17/cursor-metrics) (MIT).

Shows Cursor plan usage in the status bar for current spend-based plans (Ultra / Pro):

- Plan name (e.g. `Ultra`)
- Included plan usage / First-party models / API percentages
- On-demand (extra) spend

Legacy request-quota plans still show included requests when that is what the API returns.

**Extension ID:** `tomippe.cursor-usage`

### Upstream

Original project: [Cursor Usage / cursor-metrics](https://github.com/wrick17/cursor-metrics) by Pratyush Poddar (wrick17). This fork keeps the MIT license and original copyright notice.

Not affiliated with Anysphere / Cursor.

### Commands

- `Cursor Usage: Open Dashboard`
- `Cursor Usage: Show Details`
- `Cursor Usage: Refresh`

### Settings

Same `cursorUsage.*` keys as upstream (uninstall the original extension if both would conflict).

### License

MIT — see `LICENSE`.

## 日本語

**公式ページ:** [apps.tomippe.jp/cursor-usage](https://apps.tomippe.jp/cursor-usage/)  
リポジトリ: [tomippe/cursor-usage](https://github.com/tomippe/cursor-usage)

[wrick17/cursor-metrics](https://github.com/wrick17/cursor-metrics) のフォーク（MIT）。

Spendベースのプラン（Ultra / Pro）向けに、Cursorのプラン使用量をステータスバーに表示します。

- プラン名（例: `Ultra`）
- 定額分 / 純正モデル / API の消化率
- 追加従量分の支出

従来のリクエスト枠プランでは、APIが返す定額内リクエストも表示します。

**拡張機能 ID:** `tomippe.cursor-usage`

### アップストリーム

元プロジェクト: [Cursor Usage / cursor-metrics](https://github.com/wrick17/cursor-metrics)（Pratyush Poddar / wrick17）。このフォークは MIT ライセンスと元の著作権表示を維持しています。

Anysphere / Cursor とは無関係です。

### コマンド

- `Cursor Usage: ダッシュボードを開く`
- `Cursor Usage: 詳細を表示`
- `Cursor Usage: 更新`

### 設定

アップストリームと同じ `cursorUsage.*` キーです（元の拡張機能と同時に入れると競合するため、片方をアンインストールしてください）。

### ライセンス

MIT — `LICENSE` を参照。

## 中文

**官方页面:** [apps.tomippe.jp/cursor-usage](https://apps.tomippe.jp/cursor-usage/)  
仓库: [tomippe/cursor-usage](https://github.com/tomippe/cursor-usage)

[wrick17/cursor-metrics](https://github.com/wrick17/cursor-metrics) 的分支（MIT）。

在状态栏显示 Cursor 套餐使用量，面向按支出计费的套餐（Ultra / Pro）：

- 套餐名称（例如 `Ultra`）
- 定额用量 / 原厂模型 / API 使用百分比
- 按需（额外）支出

若 API 返回旧版请求配额数据，仍会显示包含的请求数。

**扩展 ID:** `tomippe.cursor-usage`

### 上游

原项目: [Cursor Usage / cursor-metrics](https://github.com/wrick17/cursor-metrics)（Pratyush Poddar / wrick17）。本分支保留 MIT 许可证与原始版权声明。

与 Anysphere / Cursor 无关联。

### 命令

- `Cursor Usage: 打开仪表盘`
- `Cursor Usage: 显示详情`
- `Cursor Usage: 刷新`

### 设置

与上游相同的 `cursorUsage.*` 配置项（若同时安装原扩展会冲突，请卸载其中一个）。

### 许可证

MIT — 见 `LICENSE`。
