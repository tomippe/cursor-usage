# Cursor Usage by tomippe

リポジトリ: [tomippe/cursor-usage](https://github.com/tomippe/cursor-usage)

[wrick17/cursor-metrics](https://github.com/wrick17/cursor-metrics) のフォーク（MIT）。

Spendベースのプラン（Ultra / Pro）向けに、Cursorのプラン使用量をステータスバーに表示します。

- プラン名（例: `Ultra`）
- 定額分 / 純正モデル / API の消化率
- 追加従量分の支出

従来のリクエスト枠プランでは、APIが返す定額内リクエストも表示します。

**拡張機能 ID:** `tomippe.cursor-usage`

## アップストリーム

元プロジェクト: [Cursor Usage / cursor-metrics](https://github.com/wrick17/cursor-metrics)（Pratyush Poddar / wrick17）。このフォークは MIT ライセンスと元の著作権表示を維持しています。

Anysphere / Cursor とは無関係です。

## コマンド

- `Cursor Usage: ダッシュボードを開く`
- `Cursor Usage: 詳細を表示`
- `Cursor Usage: 更新`

## 設定

アップストリームと同じ `cursorUsage.*` キーです（元の拡張機能と同時に入れると競合するため、片方をアンインストールしてください）。
